Gradients

CSS3 gradients let you display smooth transitions between two or more specified colors. Earlier, you had to use images for these effects. However, by using CSS3 gradients you can reduce download time and bandwidth usage. In addition, elements with gradients look better when zoomed, because the gradient is generated by the browser.

CSS3 defines two types of gradients:

  • Linear Gradients (goes down/up/left/right/diagonally)
  • Radial Gradients (defined by their center)

#Linear Gradients

To create a linear gradient you must define at least two color stops. Color stops are the colors you want to render smooth transitions among. You can also set a starting point and a direction (or an angle) along with the gradient effect.

Top to Bottom

The following example shows a linear gradient that starts at the top. It starts dark blue, transitioning to light blue:

First color
Second color
<div class="gradient-top-to-bottom"> </div>
.gradient-top-to-bottom {
    background: #0d57a7;
    background: -webkit-linear-gradient(red, yellow);
    background: -o-linear-gradient(red, yellow);
    background: -moz-linear-gradient(red, yellow);
    background: linear-gradient(red, yellow);
    height: 8em;
    width: 100%;
}

Left to Right

The following example shows a linear gradient that starts from the left. It starts dark blue, transitioning to light blue:

First color
Second color
<div class="gradient-left-to-right"> </div> .gradient-left-to-right {
    background: #0d57a7;
    background: -webkit-linear-gradient(left, #0d57a7, #409ad6);
    background: -o-linear-gradient(right, #0d57a7, #409ad6);
    background: -moz-linear-gradient(right, #0d57a7, #409ad6);
    background: linear-gradient(to right, #0d57a7, #409ad6);
    height: 8em;
    width: 100%;
}

Diagonaly

You can make a gradient diagonally by specifying both the horizontal and vertical starting positions. The following example shows a linear gradient that starts at top left (and goes to bottom right). It starts dark blue, transitioning to light blue:

First color
Second color
<div class="gradient-diagonal"> </div> .gradient-diagonal {
    background: #0d57a7;
    background: -webkit-linear-gradient(left top, #0d57a7, #409ad6);
    background: -o-linear-gradient(bottom right, #0d57a7, #409ad6);
    background: -moz-linear-gradient(bottom right, #0d57a7, #409ad6);
    background: linear-gradient(to bottom right, #0d57a7, #409ad6);
    height: 8em;
    width: 100%;
}

#Radial gradients

A radial gradient is defined by its center. To create a radial gradient you must also define at least two color stops. By default, shape is ellipse, size is farthest-corner, and position is center.

Evenly Spaced Color Stops

The following example shows a radial gradient with evenly spaced color stops:

First color
Second color
<div class="gradient-radial"> </div> .gradient-radial {
    background: #0d57a7;
    background: -webkit-radial-gradient(#409ad6, #0d57a7);
    background: -o-radial-gradient(#409ad6, #0d57a7);
    background: -moz-radial-gradient(#409ad6, #0d57a7);
    background: radial-gradient(#409ad6, #0d57a7);
    height: 8em;
    width: 100%;
}

Differently Spaced Color Stops

The following example shows a radial gradient with differently spaced color stops:

<div class="gradient-radial-colors-percentage"> </div> .gradient-radial-colors-percentage {
    background: #0d57a7;
    background: -webkit-radial-gradient(#409ad6 30%, #0d57a7 50%);
    background: -o-radial-gradient(#409ad6 30%, #0d57a7 50%);
    background: -moz-radial-gradient(#409ad6 30%, #0d57a7 50%);
    background: radial-gradient(#409ad6 30%, #0d57a7 50%);
    height: 8em;
    width: 100%;
}

Shapes

The shape parameter defines the shape. It can take the value circle or ellipse. The default value is ellipse.

The following example shows a radial gradient with the shape of a circle:

<div class="gradient-radial-colors-percentage"> </div> .gradient-radial-circle {
    background: #0d57a7;
    background: -webkit-radial-gradient(circle, #409ad6, #0d57a7);
    background: -o-radial-gradient(circle, #409ad6, #0d57a7);
    background: -moz-radial-gradient(circle, #409ad6, #0d57a7);
    background: radial-gradient(circle, #409ad6, #0d57a7);
    height: 8em;
    width: 100%;
}
