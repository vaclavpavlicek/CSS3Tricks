Shadows

With CSS3 you can add shadow to text and to elements.

In this article you will learn about the following properties:

  • box-shadow
  • text-shadow

#Text shadows

The CSS3 text-shadow property applies shadow to text. In following example, you specify the horizontal and vertical shadows with color:

CSS3 Tricks

<h2 class="text-shadow">CSS3 Tricks</h2> .text-shadow {
    font-size: 4em;
    margin: 0.5em 0;
    text-shadow: 3px 3px red;
}

Next, add a blur effect to the shadow:

CSS3 Tricks

<h2 class="text-shadow-blur">CSS3 Tricks</h2> .text-shadow-blur {
    font-size: 4em;
    margin: 0.5em 0;
    text-shadow: 3px 3px 6px red;
}

Multiple Shadows

To add more than one shadow to the text, you can add a comma-separated list of shadows.

The following example shows a red and blue neon glow shadow:

CSS3 Tricks

<h2 class="multiple-text-shadow">CSS3 Tricks</h2> .multiple-text-shadow {
    font-size: 4em;
    margin: 0.5em 0;
    text-shadow: 1px 1px 6px red, 0px 0px 12px blue;
}
