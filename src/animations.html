Animations

Some time before the most animations on web were written in jQuery. But with mobile devices came very emphasis on performance. And here comes CSS animations on stage. CSS animations allows you to gradually change styles of each element. They consist from two components - animation properties and @keyframes. animation properties allows you to set animation duration, count of repetitions and so on. @keyframes changes CSS properties during animation. For example they change element background color or position. If you want to create simple animation CSS is good for you. For complex animations use JS instead of CSS.
