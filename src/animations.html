Animations

Some time before the most animations on web were written in jQuery. But with mobile devices came very emphasis on performance. And here comes CSS animations on stage. CSS animations allows you to gradually change styles of each element. They consist from two components - animation properties and @keyframes. animation properties allows you to set animation duration, count of repetitions and so on. @keyframes changes CSS properties during animation. For example they change element background color or position. If you want to create simple animation CSS is good for you. For complex animations use JS instead of CSS.

#CSS animation

To animate some element use animation property or its sub-properties (duration, timing, interation count, ...). Remember animation does not configure animation appearance. To do this use @keyframes.

Here are some sub-properties:

  • animation-name - Declares the name of the @keyframes which will be applied.
  • animation-duration - Duration of one animation cycle.
  • animation-delay - Time between the element being loaded and the start of the animation.
  • animation-iteration-count - The number of times animations will be performed.
  • animation-play-state - Plays or pauses the animation.

#CSS @keyframes

The @keyframes lets you control each step of animation by creating keyframes (or waypoints) along the animation sequence that must be reached by certain points during the animation. To use keyframes, you create a @keyframes rule with a name that is then used by the animation-name property to match an animation to its keyframe list. Each @keyframes rule contains a style list of keyframe selectors, each of which comprises a percentage along the animation at which the keyframe occurs as well as a block containing the style information for that keyframe.

#Movement

This is an easy example of animation which moves with some element. Create div with class called square. In this class set background color to red, position to relative, place to upper left hand corner and set its height and width to 200px. Now we have static red square. Lets create moving animation.

<div class="square"> </div>

Create animation using @keyframes and call it move-from-top-to-bottom-and-back. At the beginning set top to 0 and left to 0. At 50% set top to 400px and left to 0px. At the end of animation set top and left back to 0. Back in square class set animation to move-from-top-to-bottom-and-back and animation-duration to 3s. Finally you can set animation-iteration-count to infinite and this animation will never stop.

.square {
    background-color: red;
    position: relative;
    top: 0px;
    left: 0px;
    height: 200px;
    width: 200px;
    -webkit-animation: move-from-top-to-bottom-and-back;
    -webkit-animation-duration: 3s;
    -webkit-animation-iteration-count: infinite;
    animation: move-from-top-to-bottom-and-back;
    animation-duration: 3s;
    animation-iteration-count: infinite;
}
@-webkit-keyframes move-from-top-to-bottom-and-back {
    0% {
        top: 0;
        left: 0;
    }
    50% {
        top: 400px;
        left: 0;
    }
    100% {
        top: 0;
        left: 0;
    }
}
@keyframes move-from-top-to-bottom-and-back {
    0% {
        top: 0;
        left: 0;
    }
    50% {
        top: 400px;
        left: 0;
    }
    100% {
        top: 0;
        left: 0;
    }
}

#Sizing

In this example we'll resize square with animations. Create div with class square as you did in example above. Set background color to red, height and width to 200px, position to relative, top and left to 0. Now we have our square and lets do some animation.

<div class="square"> </div>

First of all create animation called resize. At 50% square will be shrinked to 10px. Don't forget to move square about 95px from top and left. At the end of the animation cycle set top and left to 0 and height and width to 200px. Now we have animation lets assign it to square. You can use only animation property with more params instead of animation sub-properties. Simply assign to square animation name, duration, timing function and count of repetitions.

.square {
    -webkit-animation: resize 2s linear infinite;
    animation: resize 2s linear infinite;
    background-color: red;
    position: relative;
    top: 0px;
    left: 0px;
    height: 200px;
    width: 200px;
}

@-webkit-keyframes resize {
    50% {
        height: 10px;
        width: 10px;
        left: 95px;
        top: 95px;
    }
    100% {
        height: 200px;
        width: 200px;
        left: 0;
        top: 0;
    }

@keyframes resize {
    50% {
        height: 10px;
        width: 10px;
        left: 95px;
        top: 95px;
    }
    100% {
        height: 200px;
        width: 200px;
        left: 0;
        top: 0;
    }
}

#Rotation

In the last example we'll rotate with our square. Create div with class square and set its background color to red, height and width to 200px, and margin to 100px 0 100px 17px. We have our square ready so lets create animation.

<div class="square"> </div>

We can create animation on square hover. First create animation rotate using @keyframes and at the end of animation set transform to rotate 360 degrees. At last assign it to square on hover using animation property.

.square {
    background-color: red;
    top: 0px;
    left: 0px;
    height: 200px;
    width: 200px;
    margin: 100px 0 100px 17px;
}
.square:hover {
    -webkit-animation: rotate 2s linear infinite;
    animation: rotate 2s linear infinite;
}
@-webkit-keyframes rotate {
    100% {
        -webkit-transform: rotate(360deg);
    }
}
@keyframes rotate {
    100% {
      transform: rotate(360deg);
    }
}
